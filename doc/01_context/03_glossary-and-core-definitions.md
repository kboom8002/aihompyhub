# Glossary / Core Definitions

## AI Homepage
인간 고객용 홈페이지이자, AI가 공식 정답으로 채택하기 쉬운 구조를 가진 웹사이트

## Brand SSoT
브랜드의 공식 정답 본점

## Media SSoT
질문, 해설, 사례, 비교, 사용 맥락을 구조화하는 허브

## Answer Commerce
질문에 대한 공식 답변에서 루틴, 상담, 구매로 이어지는 전환 구조

## Tenant
개별 브랜드 또는 개별 브랜드 계정/조직 단위

## Factory
여러 브랜드의 AI Homepage를 반복 생산·운영·개선하는 운영체계

## Meta Model
모든 tenant가 공유하는 공통 구조 모델

## Template
디자인 시스템 + 콘텐츠 시스템 + 웹사이트 구조 + trust grammar + conversion grammar의 조합

## Generator
brand brief, 제품 데이터, 레퍼런스를 입력받아 ontology / content / structure 초안을 만드는 도구

## Observatory
측정, 이상 탐지, root cause 분석, fix-it을 수행하는 운영 계층

## Question Cluster
유사한 질문을 묶은 운영 단위

## AnswerCard / Routine / Compare
정답, 실행 루틴, 선택 비교를 담당하는 canonical object

## TrustBlock / Evidence / BoundaryRule
신뢰 노출, 근거, 적용 범위/주의를 담당하는 trust object
