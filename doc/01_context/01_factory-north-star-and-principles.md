# Factory North Star / Principles

## North Star
여러 스킨케어/프리미엄 B2C 브랜드가 **질문 자산을 정본으로 만들고, 신뢰 검수를 거쳐, AI와 인간 모두가 채택하기 쉬운 공식 정답 웹으로 반복 생산·운영·개선**할 수 있는 Factory OS를 만든다.

## 핵심 미션
- Brand SSoT 구축
- Media SSoT 확장
- Answer Commerce 연결
- Search / GEO / Trust / Observatory의 내재화
- Multi-tenant 반복 생산성 확보

## 10개 원칙
1. Question-first
2. Answer-first
3. Graph, not tree
4. Object-based, not page-based
5. Search is a core feature
6. Trust always visible
7. Compare is first-class
8. Routine is above product for action intent
9. Template는 운영 문법 조합
10. Automation은 draft까지만

## 운영 루프
질문 자산 → 정본 → 측정 → 원인 → 조치

## 성공 정의
- priority question coverage 증가
- trust-complete live object 증가
- search resolution / GEO coverage 개선
- no-dead-end ratio 개선
- alert → RCA → fix-it recovery 루프 작동
