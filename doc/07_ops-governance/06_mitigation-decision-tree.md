# Mitigation Decision Tree

## mitigation types
- monitor only
- limited live
- trust restriction
- GEO exclusion
- search unindex
- publish block
- rollback
- suspend
- lane pause

## core rules
- trust-critical 먼저
- search/GEO-only 문제는 site 전체 suspend 금지
- mitigation 후 RCA / fix-it / revalidation 필수
