# TASKFLOW Module Cards v2

## 사용법
이 문서는 빠른 모듈 전환, 모듈 추천, starter routing, save seed 설계, scenario 선택을 위한 카드 모음이다.

각 카드는 아래 항목을 가진다.
- module_id
- module_name
- purpose
- target_focus
- stage_scope
- representative_error
- primary_intervention
- starter_titles
- starter_fit_signals
- own_case_conversion_rule
- challenge_fit
- assetization_point
- suitable_outputs
- save_seed
- recommended_next_module

---

## M0 카드
- module_id: M0
- module_name: AI 기초 활용 리터러시와 TASKFLOW 입문
- purpose: 대화의 환상에서 벗어나 구조, 환경, 도구를 쥐기
- target_focus: orientation
- stage_scope: 오리엔테이션 / 본편 진입 전환
- representative_error: 도구·태도·환경 혼동
- primary_intervention: 핀셋 질문
- starter_titles: 회의 정리형 작업 / 독자가 중요한 작업 / 공지·안내형 작업
- starter_fit_signals: 최근 작업이 흔들림 / 새 세션 필요 여부 불명확 / 무엇부터 시작할지 모름
- own_case_conversion_rule: starter로 새 세션 여부와 T/O/K/W를 먼저 쓰게 한 뒤 실제 작업으로 전환
- challenge_fit: 초급 진입
- assetization_point: 세션 위생 체크리스트, 도구 선택 기준
- suitable_outputs: 작업 무게 판단표, 도구 선택 메모
- save_seed: session_split_needed / first_missing_block / recommended_next_module
- recommended_next_module: M1, M2, M3, M4
