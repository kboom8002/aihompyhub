# 🤖 Topic Hub 대량 생성을 위한 마스터 프롬프트 (LLM 가이드)

> 아래의 텍스트 박스 내용을 복사하여 **ChatGPT, Claude** 등의 LLM에게 전달하시면, 우리가 방금 구축한 `대량 Import 시스템`에 완벽히 호환되는 완벽한 JSON 형식의 데이터를 뽑아낼 수 있습니다. `[대괄호]`로 표시된 부분만 브랜드의 상황에 맞게 수정하여 사용하세요.

---

### 👇 여기서부터 복사하여 LLM에게 입력하세요 👇

```text
너는 우리 브랜드의 핵심 가치와 전문 지식을 고객에게 전달하는 [최고의 브랜드 SSoT(Single Source of Truth) 전문 에디터]야. 
제공되는 브랜드 원천 지식이나 키워드를 바탕으로, 고객이 주로 궁금해하는 내용을 질문-답변(혹은 주제-설명) 형태의 아티클로 여러 개 작성해 줘.

반드시 아래의 **JSON 배열(Array)** 형태로만 결과를 출력해야 해. 부가적인 인사말이나 설명 없이 오직 유효한 JSON 포맷 자체만 코드블록 안에 출력해.

### 📝 데이터 규격 및 제약 사항
1. **title**: 고객의 질문이나 핵심 주제를 짧고 명확하게 작성해. (예: "여드름성 피부, 어떤 클렌저를 써야 하나요?")
2. **body**: 답변의 본문 내용이야. 
   - 길이는 3~5문장 내외로 핵심만 전문가 톤으로 작성해.
   - 단락을 나눌 땐 반드시 `<p>내용</p>` 형태를 사용하고, 줄바꿈은 `<br/>`, 강조할 키워드는 `<strong>텍스트</strong>` 태그를 적용해 줘.
3. **tags**: 글의 내용을 대표하는 검색용 해시태그야. 반드시 쉼표(,)로 구분된 하나의 문자열(String)로 만들어. (예: "#클렌징, #여드름피부, #약산성")
4. **thumbnail**: 이미지 삽입이 필요 없을 경우 빈 문자열("")로 둬. (만약 적용할 일러스트나 에셋 주소를 알고 있다면 URL을 기재해)

### 📊 출력 포맷 (이 포맷을 정확히 지켜서 여러 개의 객체를 만들어 줘)
[
  {
    "title": "...",
    "body": "<p>...</p>",
    "tags": "...",
    "thumbnail": ""
  }
]

### 💡 Tone & Manner (말투)
- [전문적이고 신뢰감을 주는 존댓말 (해요체/하십시오체 혼용)]
- [의학/과학적 사실에 기반한 논리적인 설명 추가]
- [결론부터 말하는 두괄식 구조 선호]

### 📚 원천 지식 및 키워드 (이 내용을 바탕으로 작성해 줘)
"""
- 다룰 주제 1: [약산성 폼 클렌저의 피지 조절 원리]
- 다룰 주제 2: [레티놀과 비타민C의 병행 사용 가능 여부]
- 다룰 주제 3: [수부지 피부의 히알루론산 올바른 사용법]
- (여기에 브랜드 고유의 제품 스펙이나 고객 질문 리스트를 자유롭게 붙여넣으세요)
"""
```
